title: How to execute steps on failure?
author: bradrydzewski
weight: 1
draft: true
---


```
kind: pipeline
type: digitalocean
name: default

steps:
- name: foo
  commands:
  - echo foo
  - exit 1

- name: bar
  commands:
  - echo bar
  when:
    status: [ failure ]
```
